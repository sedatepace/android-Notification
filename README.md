# Android Notification

## Introduce

안드로이드에서 카톡앱처럼 알림기능을 통해 사용자에게 정보전달을 해주고 싶을 때가 있다.  안드로이드에서는  Notification 이라는 기능을 통해서 스마트폰에 알림을 보내줄 수 있다. 

여기에서는 특정 시간, 또는 주기마다 Foreground, Background에서 Notification 기능을 통해 앱에 알림을 띄워보려고 한다.
